# ESSS_Day1
## Important Notice

The repositories provided in this organization contain the R scripts and supporting materials used during the Summer School. They are **not complete RStudio projects** and do not necessarily include all files required to reproduce the development environment used by the instructors.

Participants are expected to:

* Create their own local RStudio project if desired;
* Install the required R packages listed in the materials;
* Adapt file paths and project settings to their own environment.

The repositories are intended as educational resources and examples rather than ready-to-run, fully configured software projects.
